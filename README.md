App de Organização de Finanças Pessoais

Esse projeto foi pensado para ajudar pessoas a organizarem melhor suas finanças do dia a dia.
A ideia é permitir o controle de gastos e ganhos mensais de forma simples.

O aplicativo ajudaria o usuário a ter mais clareza sobre onde está gastando seu dinheiro
e a se planejar melhor financeiramente.

Esse projeto faz parte de um desafio educacional com foco em planejamento e uso de IA
para apoiar a criação de soluções digitais.
